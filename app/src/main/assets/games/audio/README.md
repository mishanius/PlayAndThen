# Shared Audio Files

This folder contains audio files shared across all TS games.

## Setup (Manual Step Required)

Copy audio files from `res/raw/` to this folder:

```bash
# From project root
cp app/src/main/res/raw/*.ogg app/src/main/assets/games/audio/
```
